# How to Install Android Studio & Run Your Quran Academy App

## Step 1: Download Android Studio

1. Go to: **https://developer.android.com/studio**
2. Click **"Download Android Studio Hedgehog"**
3. Save the `.exe` file to your computer

## Step 2: Install Android Studio

1. **Run the `.exe` file** you downloaded
2. Click **"Next"** → **"Next"** → **"Install"**
3. Wait 5-10 minutes for installation
4. Click **"Finish"** when done

## Step 3: Initial Setup (First Time Only)

1. When Android Studio opens, choose **"Do not import settings"** → Click OK
2. Choose **"Standard"** installation type → Click Next
3. Make sure **"Android SDK"** is checked → Click Next
4. Accept all license agreements → Click Finish
5. Wait for SDK download (10-15 minutes)

## Step 4: Open Your Project

1. Click **"Open"** on the welcome screen
2. Navigate to: `C:/Users/Atif Ahmed/OneDrive/Desktop/quran acc`
3. Click **OK**
4. **Wait for Gradle sync** (bottom right corner shows progress)

## Step 5: Run the App

**Option A: On Your Phone**
1. Connect your Android phone via USB
2. Enable **USB Debugging** on your phone (Settings → Developer Options)
3. Click the **green ▶️ Run button** in Android Studio toolbar

**Option B: On Emulator**
1. Click **Device Manager** (right sidebar)
2. Click **"Create Virtual Device"**
3. Select **"Pixel 5"** → Click Next
4. Download system image if needed
5. Click **Play ▶️** button next to device

## System Requirements

- Windows 10/11 (64-bit)
- 8GB RAM minimum (16GB recommended)
- 10GB disk space
- Java JDK 17 (installed by Android Studio)
